## Content Production Workflow

For each page:
1. **Outline** — write H1/H2 structure, confirm keyword targets
2. **Draft** — write full content (can use AI assistance for first draft)
3. **Original assets** — screenshots, code samples, diagrams
4. **SEO review** — verify title/meta/schema/internal links/FAQ section
5. **Publish** — deploy, submit to Indexing API immediately
6. **Monitor** — check GSC indexing status within 48 hours
